# Markdown 语法指南

以下是常用的 Markdown 格式示例：

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
...
```

## 列表

### 无序列表

```markdown
- 项目1
- 项目2
- 项目3
```

### 有序列表

```markdown
1. 项目1
2. 项目2
3. 项目3
```

## 强调和斜体

```markdown
*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
```

## 链接和图片

### 链接

```markdown
[链接文本](http://example.com)
```

### 图片

```markdown
![图片描述](http://example.com/image.jpg)
```

## 引用

```markdown
> 引用文本
```

## 代码块

~~~markdown
```语言
代码块
```
~~~

## 表格

```markdown
| 列1 | 列2 |
|-----|-----|
| 内容1 | 内容2 |
| 内容3 | 内容4 |
```

## 水平线

```markdown
---
```

## 换行标签
```markdown
</br>
```
